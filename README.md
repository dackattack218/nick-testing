# Nick2

## Getting Started

1. Clone the repository
```bash
git clone https://github.com/dackattack218/nick2.git
cd nick2
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

## Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Shadcn UI
- React Router DOM
